# Velvet & Vine Salon ✂️✨

An upscale, luxury landing page and booking system built for **Velvet & Vine Salon**. This application offers a high-end user experience, dynamic service menus, and a streamlined client-booking workflow that redirects requests directly to the business via WhatsApp.

## 🚀 Features

- **Luxury Aesthetic:** Premium UI layout tailored for high-end boutique styling and lifestyle branding.
- **Dynamic Multi-Page Routing:** Built utilizing Next.js App Router to separate specific category menus (`/services/custom-color`, `/services/extensions`, etc.).
- **Smooth Animations:** Fully interactive page transitions and popups engineered via Framer Motion.
- **Smart Booking Modal:** Captured client details automatically pre-fill custom-encoded direct communication logs.
- **WhatsApp Integration:** Bypasses dense cloud-database maintenance for immediate service alignment by generating direct click-to-chat bookings.

## 💻 Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Components:** shadcn/ui
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel
